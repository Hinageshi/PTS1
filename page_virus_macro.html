Les Virus Macro

Cible du virus

Les vecteurs de contamination des virus macro sont les documents contenant des macros dans Word et Excel, plus précisément le code exécutable contenu dans les macros.

Principe de Fonctionnement

Les applications telles que Word ou Excel contiennent un langage de programmation qui permet d'automatiser des opérations complexes à l'aide de macroinstructions enregistrées dans les documents. De ce fait, ces documents contiennent du code exécutable qui permettent à certains virus écrits en langage macro d'être exécutés dans les documents et d'infecter l'application et tous les documents, puis, dans certains cas, infecter le système d'exploitation.
